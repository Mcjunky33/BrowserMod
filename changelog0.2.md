# Changelog

## [0.2] – Update

### English

- **Bug Fixes:** 
- Video players are now working correctly.
- In PIP mode, the tooltips are only displayed at the bottom when hovering.
- Open Browser Key and toggle PIP key now only work in important screens. This prevents the browser from being unintentionally opened via a key in modded screens.


- **New:**
- Added Support for [MCEF [Keksuccino's Fork]](https://modrinth.com/mod/mcef-keksuccino) ver. 2.2.0
- Added Help/Credits Html.
